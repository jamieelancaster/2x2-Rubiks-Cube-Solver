Implementation of algorithms to solve the 2x2 Rubik's cube. 

Currently I have the base cube mechanics and GUI implemented along with a Breadth-First-Search algorithm to create a heuristic lookup table. 

I have plans to improve the database structure of the lookup table, as well as minorly improving the speed to the BFS algorithm.
After that I also plan on implementing:
A*
IDA*
as well as more exhaustive search methods
